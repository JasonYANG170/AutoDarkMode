# YANG-Auto-Dark-Mode
YANG Auto Dark Mode 自动化深色模式程序项目启动

于2023年1月26日发布V1.0.0测试版本

现已发布V2.0.0正式版本

解决了V1.0.0Beta到V1.5.0Beta中的所有BUG。

现在程序全部功能已经可以正常使用了。

暂未发现新的BUG，如有遇到请告知我。
# 使用说明
输入设定时间（24小时制）后深色模式将会在大于等于设定时间时启动。

为确保程序稳定运行，请选择程序后台运行按钮。
