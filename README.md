## 功能
- ✅ 定时切换深色模式
- ✅ 定时切换浅色模式
- ✅ 开机自启动
- ✅ 后台运行

**现项目功能合并至"健康守护"项目**  
现在程序全部功能已经可以正常使用了。  
暂未发现新的BUG，如有遇到请告知我。
# 使用说明
输入设定时间（24小时制）后深色模式将会在大于等于设定时间时启动。

为确保程序稳定运行，请选择程序后台运行按钮。
